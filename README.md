# Drafted plugin

Public mirror of the Claude Code plugin and Gemini CLI extension for [Drafted](https://drafted.live).

Source of truth: the private `ddfourtwo/drafted` repo. Do not PR here — open issues at https://drafted.live.

## Install

**Claude Code:**

```
claude plugin marketplace add ddfourtwo/drafted-plugin
claude plugin install drafted@drafted
```

**Gemini CLI:**

```
gemini extensions install https://github.com/ddfourtwo/drafted-plugin
```

See the full install flow at https://drafted.live/install.
